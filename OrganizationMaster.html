Note-Taking Methods

If you are struggling to find a well sorted and easy note-taking method, well you have come to the right place. Here some methods that might be useful in note taking for any subject.

This is called the Cornell method. The cornell method is very effective in terms of organizing and review information. It allows you to write down key points as well as leave room for summaries, questions, and comments to help recall.

    The benifits of this method:
  • Easy to review and recall in question and answer format
  • Allows engagement during lectures and active listening.
  • Best for identifying key ideas and concepts
      • The disadvantages of this method:
  • Takes more time to divide and draw in the grid
  • Its hard fit a lot of notes which might then require some adjustments with extending or using multiple pages to create the format.

    • This is called the T-Notes Method. The T-Notes method is a method you can use when you need to write down quick notes of a lecture in math, discussions, or reading.

      The benefits of this method:
  • Helps organise large amounts of information
  • Easy to review and recall the notes
  • Efficient in terms of adding new information or adding to what's already written.
      • The disadvantages of this method:
  • It takes a bit of time to set up the grid system
  • Not suitable for subjects that require extensive information and details
